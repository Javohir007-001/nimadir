Flask bilan Backend dunyosiga kiring

Flask — bu Python tilida yozilgan eng mashhur veb-microframework. U yengil, tez va o'rganish uchun juda qulay. Agar siz veb-dasturlashni endi boshlayotgan bo'lsangiz, Flask siz uchun eng to'g'ri tanlovdir.

Bu darslik sizga Flaskning asosiy poydevorini 15 daqiqa ichida o'rgatish uchun mo'ljallangan.

2

Muhitni sozlash va o'rnatish

Avval virtual muhit (virtual environment) yarating, so'ngra Flaskni terminal orqali o'rnating:

terminal
pip install flask
3

Sizning birinchi "Hello World"ingiz

Yangi app.py faylini oching va quyidagi kodni kiriting:

app.py 
from flask import Flask

app = Flask(__name__)

@app.route("/")
def index():
    return "Salom, Flask darslariga xush kelibsiz!"

if __name__ == "__main__":
    app.run(debug=True)

Bu nima qiladi?

  • Flask(__name__) - ilovaning asosiy ob'ektini yaratadi.
  • @app.route("/") - foydalanuvchi asosiy sahifaga kirganini nazorat qiladi.
  • debug=True - kodni o'zgartirganda server o'zi yangilanishini ta'minlaydi.
4

Dinamik Routing (Yo'naltirish)

URL orqali ma'lumot qabul qilish juda oson: 

@app.route("/user/<name>")
def greet(name):
    return f"Salom, {name}!"
